FAQ

Häufige Fragen

Brauche ich einen eigenen API-Key?

Ja. CE365 Agent nutzt KI-APIs im BYOK-Modell (Bring Your Own Key). Du zahlst die API-Kosten direkt beim Provider — typisch 1–5 € pro Monat bei normaler Nutzung. Unterstützt werden Anthropic (Claude), OpenAI (GPT) und OpenRouter.

Welche Betriebssysteme werden unterstützt?

CE365 Agent läuft auf Windows, macOS und Linux. Die Diagnose- und Repair-Tools nutzen jeweils die nativen System-Tools der Plattform.

Werden Kundendaten in die Cloud geschickt?

Nein. Alle Daten bleiben lokal auf dem Techniker-PC. Nur anonymisierte Problembeschreibungen werden an den KI-Provider gesendet. Personenbezogene Daten werden vorher automatisch erkannt und entfernt (PII Detection via Microsoft Presidio).

Kann CE365 Schaden am Kunden-PC anrichten?

CE365 führt keine Aktionen ohne deine explizite Freigabe aus. Reparaturen werden vorgeschlagen und müssen mit GO REPAIR bestätigt werden. Jede Aktion wird protokolliert und kann nachvollzogen werden.

Was bedeutet “pro Seat”?

Ein Seat = ein gleichzeitig aktiver Techniker. Wenn dein Team 3 Techniker hat, die CE365 gleichzeitig nutzen, brauchst du 3 Seats. Nicht aktive Techniker belegen keinen Seat.

Kann ich jederzeit kündigen?

Jederzeit. Das Abo läuft monatsweise ohne Mindestlaufzeit. Nach Kündigung bleibt die Lizenz bis zum Ende des bezahlten Zeitraums aktiv.
