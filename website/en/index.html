FAQ

Frequently asked questions

Do I need my own API key?

Yes. CE365 Agent uses AI APIs in a BYOK model (Bring Your Own Key). You pay API costs directly to the provider — typically €1–5 per month with normal usage. Supported: Anthropic (Claude), OpenAI (GPT), and OpenRouter.

Which operating systems are supported?

CE365 Agent runs on Windows, macOS, and Linux. The diagnostic and repair tools use each platform's native system tools.

Is customer data sent to the cloud?

No. All data stays local on the technician's PC. Only anonymized problem descriptions are sent to the AI provider. Personal data is automatically detected and removed before transmission (PII Detection via Microsoft Presidio).

Can CE365 damage a client's PC?

CE365 never executes actions without your explicit approval. Repairs are suggested and must be confirmed with GO REPAIR. Every action is logged and traceable.

What does "per seat" mean?

One seat = one concurrently active technician. If your team has 3 technicians using CE365 simultaneously, you need 3 seats. Inactive technicians don't occupy a seat.

Can I cancel anytime?

Yes. The subscription runs monthly with no minimum term. After cancellation, the license remains active until the end of the paid period.
